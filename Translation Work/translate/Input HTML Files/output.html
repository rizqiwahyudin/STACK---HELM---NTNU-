It will be worth keeping in mind that \(\coth(z)=\dfrac{\cosh(z)}{\sinh(z)}\). We start by rewriting the expression in terms of \(\sinh({@aa@}{@xx@})\) and \(\cosh({@aa@}{@xx@})\) as at least these are more like the expression we are targeting. For the numerator we use \[e^z=\cosh(z)+\sinh(z)\] while for denominator we can notice that this is already quite close to \(\sinh({@aa@}{@xx@})\) indeed \[ \sinh({@aa@}{@xx@})=\frac{e^{{@aa@}{@xx@}}-e^{-{@aa@}{@xx@}}}{2}.\] Det vil være verdt å huske på at \ (\ coth (z) = \ dfrac {\ cosh (z)} {\ sinh (z)} \).Vi starter med å omskrive uttrykket i form av \ (\ sinh ({@aa@} {@xx@}) \) og \ (\ cosh ({@aa@} {@xx@}) \) som i det minste disseer mer som uttrykket vi retter oss mot.For telleren bruker vi \ [e^z = \ cosh (z)+\ sinh (z) \] mens for nevner kan vi merke at dette allerede er ganske nær \ (\ sinh ({@aa@} {@xx@}) \) faktisk \ [\ sinh ({@aa@} {@xx@}) = \ frac {e^{{@aa@} {@xx@}}-e^{-{@aa@}{@xx@}}} {2}. \]

Note that instead of spotting this we could just write each \(e^z=\cosh(z)+\sinh(z)\), there would be some cancelation after using the even / odd properties of \(\cosh(z)\) and \(\sinh(z)\). The result is of course the same. It's good exercise to confirm.
